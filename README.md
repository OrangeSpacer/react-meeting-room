# Форма для бронирования переговорной

## Функции
- Валидация полей
- Возможность отправить форму(результат появляется в консоле)
- Возможность очистки формы
- Адаптив до 320px

## Стек
- React(использовался js)
- react-hook-form
- react-select
- classnames
- css
- Сборщиком проекта является vite
